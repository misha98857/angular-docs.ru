# Angular Russian Glossary

Этот глоссарий должен использоваться всеми ИИ-агентами для обеспечения единообразия перевода.

| Observable                    | Observable (или "Поток")                       | В профессиональной среде чаще используют "Observable". Для новичков допустимо "Поток", но лучше придерживаться оригинала или "Observable-поток". |

| Directive                     | Директива                                      |                                                                                                                                                  |

| Template                      | Шаблон                                         |                                                                                                                                                  |

## Стиль кода и примеров

- **Переменные:** Не переводить (`const user = ...` -> `const user = ...`).
- **Комментарии в коде:** Переводить только если они объясняют суть обучающего примера.
- **Консольный вывод:** Не переводить, если это лог ошибки.

## Общие правила

- Обращение к читателю: "Вы" (с маленькой буквы) или безличные конструкции ("Следует использовать...", "Компонент
  позволяет...").
- Избегать машинного перевода ("Является используемым..."). Писать живым языком.
