# Неиспользуемые импорты в Standalone-компонентах

Эта диагностика обнаруживает случаи, когда массив `imports` в `@Component` содержит символы, которые не используются
внутри шаблона.

```typescript
@Component({
  imports: [UsedDirective, UnusedPipe],
})
class AwesomeCheckbox {}

```

## В чем проблема?

Неиспользуемые импорты создают лишний шум в коде и могут увеличить время компиляции.

## Что следует сделать?

Удалите неиспользуемый импорт.

```typescript
@Component({
  imports: [UsedDirective]
})
class AwesomeCheckbox {}
```

## Что делать, если этого нельзя избежать?

Эту диагностику можно отключить, отредактировав файл `tsconfig.json` проекта:

```json

{
  "angularCompilerOptions": {
    "extendedDiagnostics": {
      "checks": {
        "unusedStandaloneImports": "suppress"
      }
    }
  }
}
```

Смотрите раздел [конфигурация расширенной диагностики](extended-diagnostics#configuration) для получения дополнительной
информации.
