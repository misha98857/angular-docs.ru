# Корневой элемент не найден

Компоненты для начальной загрузки определяются в свойстве `bootstrap` декоратора `@NgModule` или передаются первым
параметром в `bootstrapApplication` для standalone-компонентов.

Эта ошибка возникает, когда Angular пытается выполнить начальную загрузку одного из этих компонентов, но не может найти
соответствующий ему узел в DOM.

## Отладка ошибки

Эта проблема возникает, если селектор не совпадает с тегом

```typescript
@Component({
  selector: 'my-app',
  ...
})
export class AppComponent {}
```

```angular-html
<html>
    <app-root></app-root> <!-- Не совпадает с селектором -->
</html>
```

Замените тег на правильный:

```angular-html
<html>
    <my-app></my-app> <!-- OK -->
</html>
```
