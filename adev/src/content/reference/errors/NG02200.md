# Отсутствует Iterable Differ

`NgFor` не удалось найти механизм сравнения (iterable differ) для переданного значения. Убедитесь, что это итерируемый
объект, например `Array`.

## Отладка ошибки

При использовании `ngFor` в шаблоне необходимо использовать какой-либо итерируемый тип (Iterable), например `Array`,
`Set`, `Map` и т.д.
Если вы пытаетесь перебрать ключи объекта, обратите внимание на [KeyValue pipe](/api/common/KeyValuePipe).
