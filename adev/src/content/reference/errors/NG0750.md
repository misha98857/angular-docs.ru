# Не удалось загрузить зависимости @defer

Эта ошибка возникает, когда не удается загрузить зависимости для блока `@defer` (обычно из-за проблем с сетью), и при
этом не настроен блок `@error` для обработки сбоя. Отсутствие блока `@error` в такой ситуации может негативно сказаться
на пользовательском опыте.

## Отладка ошибки

Убедитесь, что вы добавили блоки `@error` в свои блоки `@defer` для обработки состояний ошибки.
