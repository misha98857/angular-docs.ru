# Некорректный multi-провайдер

Angular выбрасывает эту ошибку во время выполнения, когда внедряет токен, предназначенный для использования с
`multi: true`, но вместо массива находит значение другого типа. Например, `ENVIRONMENT_INITIALIZER` следует
предоставлять следующим образом: `{provide: ENVIRONMENT_INITIALIZER, multi: true, useValue: () => {...}}`.
