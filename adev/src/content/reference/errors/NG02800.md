# Поддержка JSONP в конфигурации HttpClient

Angular генерирует эту ошибку при попытке выполнить запрос `JSONP` без включения его поддержки в конфигурации
`HttpClient`.
Чтобы включить поддержку `JSONP`, выполните одно из следующих действий:

- добавьте `withJsonpSupport()` в качестве аргумента при вызове функции `provideHttpClient` (например,
  `provideHttpClient(withJsonpSupport())`), если используется `bootstrapApplication`;
- импортируйте `HttpClientJsonpModule` в корневой `AppModule`, если используется bootstrap на основе NgModule.

## Отладка ошибки

Убедитесь, что поддержка JSONP включена в приложении: либо через вызов функции `withJsonpSupport` (при использовании
`provideHttpClient`), либо через импорт модуля `HttpClientJsonpModule`, как описано выше.
