# Экспорт не найден

<docs-video src="https://www.youtube.com/embed/fUSAg4kp2WQ"/>

Angular не может найти директиву с именем экспорта `{{ PLACEHOLDER }}`. Имя экспорта указывается в свойстве `exportAs`
декоратора директивы. Это часто происходит при использовании `FormsModule` или модулей Material в шаблонах, если вы
забыли импортировать соответствующие модули.

HELPFUL: Это ошибка времени выполнения, аналогичная распространенной ошибке
компилятора [NG8003: No directive found with export](errors/NG8003).

## Отладка ошибки

Используйте имя экспорта, чтобы найти шаблоны или модули, использующие этот экспорт.

Убедитесь, что все зависимости правильно импортированы и объявлены в ваших NgModule. Например, если не найден экспорт
`ngForm`, для устранения ошибки необходимо импортировать `FormsModule` и добавить его в список imports в файле
`*.module.ts`.

```ts

import { FormsModule } from '@angular/forms';

@NgModule({
…
imports: [
FormsModule,
…

```

Если вы недавно добавили импорт, возможно, потребуется перезапустить сервер, чтобы изменения вступили в силу.
