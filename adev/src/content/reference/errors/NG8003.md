# Отсутствует цель ссылки

<docs-video src="https://www.youtube.com/embed/fUSAg4kp2WQ"/>

Angular не может найти директиву с экспортным именем `{{ PLACEHOLDER }}`.
Обычно это связано с отсутствием импорта или свойства [`exportAs`](api/core/Directive#exportAs) у директивы.

HELPFUL: Это аналог распространенной ошибки времени выполнения [NG0301: Export Not Found](errors/NG0301), возникающий на
этапе компиляции.

## Отладка ошибки

Используйте имя ненайденного экспорта, чтобы найти шаблоны или модули, где он используется.

Убедитесь, что все зависимости правильно импортированы и объявлены в модулях.
Например, если не найден экспорт `ngForm`, необходимо импортировать `FormsModule` и добавить его в список `imports` в
файле `*.module.ts`, чтобы устранить ошибку отсутствующего экспорта.

```ts

import { FormsModule } from '@angular/forms';

@NgModule({
…
imports: [
FormsModule,
…

```

Если вы недавно добавили импорт, необходимо перезапустить сервер, чтобы изменения вступили в силу.
