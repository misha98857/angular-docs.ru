# Неверный селектор Shadow DOM

Селектор компонента, использующего `ViewEncapsulation.ShadowDom`, не соответствует требованиям к именам тегов
пользовательских элементов (custom elements).

Чтобы имя тега считалось корректным именем пользовательского элемента, оно должно:

- Быть в нижнем регистре.
- Содержать дефис.
- Начинаться с буквы \(a-z\).

## Отладка ошибки

Переименуйте селектор компонента так, чтобы он соответствовал требованиям.

**До:**

```ts
@Component({
  selector: 'comp',
  encapsulation: ViewEncapsulation.ShadowDom
…
})

```

**После:**

```ts
@Component({
  selector: 'app-comp',
  encapsulation: ViewEncapsulation.ShadowDom
…
})
```
