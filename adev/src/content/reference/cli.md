# Справочник CLI

| Команда                            | Псевдоним  | Описание                                                                                                      |
| :--------------------------------- | :--------- | :------------------------------------------------------------------------------------------------------------ |
| [`add`](cli/add)                   |            | Добавляет поддержку внешней библиотеки в ваш проект.                                                          |
| [`analytics`](cli/analytics)       |            | Настраивает сбор метрик использования Angular CLI.                                                            |
| [`build`](cli/build)               | `b`        | Компилирует приложение или библиотеку Angular в выходную директорию `dist/` по указанному пути.               |
| [`cache`](cli/cache)               |            | Настраивает постоянный дисковый кэш и получает статистику кэширования.                                        |
| [`completion`](cli/completion)     |            | Настраивает автодополнение команд Angular CLI для вашего терминала.                                           |
| [`config`](cli/config)             |            | Получает или устанавливает значения конфигурации Angular в файле `angular.json` для рабочей области.          |
| [`deploy`](cli/deploy)             |            | Запускает билдер развертывания для указанного проекта или для проекта по умолчанию в рабочей области.         |
| [`e2e`](cli/e2e)                   | `e`        | Собирает и запускает приложение Angular, затем выполняет сквозные (e2e) тесты.                                |
| [`extract-i18n`](cli/extract-i18n) |            | Извлекает сообщения i18n из исходного кода.                                                                   |
| [`generate`](cli/generate)         | `g`        | Генерирует и/или изменяет файлы на основе схемы (schematic).                                                  |
| [`lint`](cli/lint)                 |            | Запускает инструменты линтинга для кода приложения Angular в указанной папке проекта.                         |
| [`new`](cli/new)                   | `n`        | Создает новую рабочую область Angular.                                                                        |
| [`run`](cli/run)                   |            | Запускает цель Architect с опциональной пользовательской конфигурацией билдера, определенной в вашем проекте. |
| [`serve`](cli/serve)               | `s`, `dev` | Собирает и запускает ваше приложение, пересобирая его при изменении файлов.                                   |
| [`test`](cli/test)                 | `t`        | Запускает модульные тесты в проекте.                                                                          |
| [`update`](cli/update)             |            | Обновляет вашу рабочую область и её зависимости. См. https://angular.dev/update-guide/.                       |
| [`version`](cli/version)           | `v`        | Выводит версию Angular CLI.                                                                                   |
